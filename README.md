# Stroke Prediction Management System

This project is a secure web-based application for managing stroke prediction data.  
It is built using Python Flask, SQLite, and MongoDB, applying secure development practices.

## Objectives
- Secure web app for doctors to manage patient data.
- Predict stroke risk using health, demographic, and lifestyle data.
- Implement encryption, authentication, and validation.
